# CSC3916 Assignment 4

## Description
REST API with MongoDB that supports movies and reviews. Users can query movies with aggregated reviews using ?reviews=true.

## Installation
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with `DB`, `SECRET_KEY`, and `PORT`
4. Run `node server.js`

## Deployed API
https://csc3916-assignment4-bp3w.onrender.com

## Postman Collection
[Link to collection](https://www.postman.com)

## Environment Variables
- DB: MongoDB connection string
- SECRET_KEY: JWT secret key
- PORT: 8080
